Granabular: a collaborative granular synthesizer

A granular synthesizer is a sound creation device that takes recordings, chops them into tiny pieces, and then plays them back at a rapid rate to create a new sound experience. This project aims to extend this sound creation experience by enabling multiple users to control a single granular synthesizer in real-time, remotely, from their web browser.

When a user joins the session they will be assigned a single parameter to control. Each user will be given a unique parameter and once all parameters are claimed, the session will be set as occupied and no users can join. In addition to the parameter, the user also has the ability to add grains to the collective grain bucket by entering search keywords, which will locate and save related sound clips. If enabled, users will also be periodically notified that their microphone is recording so as to make some noise that will be sent to the grain bucket.

This project is open-source and available on GitHub.
